# Day 2 — Sunday: Flutter App + Full Integration

Goal: Working demo APK on Android. Camera → result → PHC SMS. GitHub pushed.

- `neoscreen_v1.tflite` trained and < 5 MB

## Morning — Task 7 (2 hrs): Flutter App Skeleton

The Flutter app is already built at `flutter/lib/`. Verify it compiles:

```bash
cd flutter
flutter pub get
flutter analyze
```

Expected: no errors. Warnings about unused imports are fine.

---

## Morning — Task 8 (2 hrs): Integrate TF Lite → UI

Copy trained model and PHC database into Flutter assets:

```bash
# From project root
python day2/setup_flutter_assets.py
```

This copies:
- `neoscreen_v1.tflite` → `flutter/assets/models/`
- `data/phc_maharashtra.db` → `flutter/assets/data/`

Then verify the tflite plugin loads correctly:

```bash
cd flutter
flutter test
```

## Afternoon — Task 9 (1 hr): Test PHC Database

```bash
# From project root
python day2/test_phc_lookup.py
```

Tests nearest-PHC lookup for 5 different GPS coordinates across Maharashtra.
Expected: each query returns a PHC name + phone + distance in < 10ms.

## Afternoon — Task 10 (1 hr): Test Twilio SMS End-to-End

```bash
# Set up your .env first
cp .env.example .env
# Edit .env and fill in TWILIO_SID, TWILIO_TOKEN, TWILIO_FROM

python day2/test_twilio.py
```

This sends a real test SMS to the nearest PHC number.
Check your Twilio console to confirm it was sent.

---

## Afternoon — Task 11 (1 hr): Full End-to-End Test (Python)

```bash
python day2/test_end_to_end.py --model neoscreen_v1.tflite
```

Simulates the full pipeline:
Image → CLAHE → sclera detect → TF Lite → classify → language output → PHC alert

---

## Evening — Task 12 (1 hr): Build APK

```bash
cd flutter
flutter build apk --release
```

APK will be at: `flutter/build/app/outputs/flutter-apk/app-release.apk`

Install on Android device:
```bash
flutter install
```

Or transfer the APK file manually to your phone and install.
